Projetos

Meus projetos até agora envolvem esse simples portfólio, pois eu estou em aprendizado. Esse projeto consiste em, a partir do HTML, desenvolver e criar sites estilizados usando o CSS3 e o HTML. Nesse projeto, tive embasamento em um exercicio proposta por uma influencer na plataforma : YouTube. Meu primeiro site, de muitos; foi desenvolvido por mim, onde eu usei a plataforma executavel Virtual Studio Code (VSCode)

Para o meu primeiro portfólio, usei 3 index CSS, de forma separada, em uma pagina HTML; com varias subpaginas. Na nav bar, onde ficam os endereçamentos para as paginas secundárias. Usei conceitos de div, onde eu defino paginas por etapas, para facilitar a estilização. Consegui fazer uma estilização concisa, e de facil visualização, tanto para os programadores; quanto para o usuário. Com um display intuitivo e chamativo.
